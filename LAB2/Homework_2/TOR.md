# บริษัท RSU ทำการว่าจ้างบริษัทที่นักศึกษาทำงานอยู่เพื่อติดตั้งระบบเครือข่ายภายในอาคาร

โดยทางองค์กรได้ให้รายละเอียดเกี่ยวกับความต้องการของระบบเครือข่ายที่ต้องทำการออกแบบและตั้งค่า switch ต่างๆ ดังนี้

### รายละเอียดในการออกแบบ

- อาคารของบริษัทมีทั้งหมด 5 ชั้น แต่ละชั้นจะมีเครื่องคอมพิวเตอร์ 10 เครื่อง
- บริษัทมีทั้งหมด 5 แผนก ได้แก่ แผนกบัญชี (Account), แผนกพนักงานขาย (Sale), แผนกวิศวกร (Engineer), แผนกบุคคล (HR) และผู้บริหาร (Executive)
- ต้องออกแบบ VLAN เพื่อรองรับแผนกทั้ง 5 แผนก เพื่อไม่ให้แต่ละแผนกมีการเชื่อมต่อกัน
- ต้องออกแบบ Private IP Address เพื่อรองรับเครื่องคอมพิวเตอร์ในแต่ละแผนก
- อุปกรณ์ switch cisco 2960 จำนวน 5 ตัว (100Mbps = 24 ports / 1Gbps = 2 ports / Console = 1)
- กำหนดให้ port GigabitEthernet 0/1 – 0/2 ทำหน้าที่เป็น Trunk Port สำหรับ switch แต่ละตัว ในแต่ละชั้น
- การเชื่อมต่อคอมพิวเตอร์ในแต่ละชั้น (โดยทำการเลือกกำหนด port บน switch ให้กับคอมพิวเตอร์ในแต่ละแผนกเอง)
  - **ชั้นที่ 1 :** Account = 5 / Sale = 0 / Engineer = 0 / HR = 4 / Executive = 1
  - **ชั้นที่ 2 :** Account = 1 / Sale = 2 / Engineer = 3 / HR = 2 / Executive = 2
  - **ชั้นที่ 3 :** Account = 3 / Sale = 0 / Engineer = 3 / HR = 3 / Executive = 1
  - **ชั้นที่ 4 :** Account = 2 / Sale = 2 / Engineer = 2 / HR = 2 / Executive = 2
  - **ชั้นที่ 5 :** Account = 5 / Sale = 2 / Engineer = 2 / HR = 0 / Executive = 1

---

**หมายเลข VLAN ที่กำหนดให้ในแต่ละแผนก**

- แผนก Account = **10**
- แผนก Sale = **20**
- แผนก Engineer = **30**
- แผนก HR = **40**
- แผนก Executive = **50**

**Range ของ Private IP Address ที่กำหนดให้ในแต่ละแผนก**

- IP Address แผนก Account = **192.168.10.1 - 192.168.10.254**
- IP Address แผนก Sale = **192.168.20.1 - 192.168.20.254**
- IP Address แผนก Engineer = **192.168.30.1 - 192.168.30.254**
- IP Address แผนก HR = **192.168.40.1 - 192.168.40.254**
- IP Address แผนก Executive = **192.168.50.1 - 192.168.50.254**
